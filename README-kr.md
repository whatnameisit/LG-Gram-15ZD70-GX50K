# LG-Gram-15ZD970-GX5YL
macOS Catalina Beta 2
## Working
  1. HD Graphics 620에 2048 MB 비디오 램을 할당했으며, HDMI가 정상 작동합니다.
  2. USB 매핑이 설정되어 있습니다.
  3. 배터리 표기가 정상 작동합니다.
  4. SMBIOS 항목을 채우면 아이메시지와 페이스타임이 정상 작동합니다.
  5. 터치패드 작동합니다.
